LD22-Alone by gamecook

This is my Ludum Dare #22 Game Source

This is a simple game using ImpactJS that was built in 48 hours. The text is a poem is by Edgar Alan Poe and game is written in Impact JS.

Dependencies

ImpactJS (http://impactjs.com)

Install

You will need a copy of Impact's src which needs to go inside of the impact/lib folder. After that simply run the Ant build script and the game should compile into a deploy folder.
